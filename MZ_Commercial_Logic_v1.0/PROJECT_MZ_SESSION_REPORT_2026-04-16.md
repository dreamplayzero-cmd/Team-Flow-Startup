## 🏛️ 1. 프로젝트 목표 및 비즈니스 로직

### 🎯 핵심 목표
- MZ세대 소상공인 창업자를 위한 상권 분석 및 성공 확률 예측
- 유동인구(인구) + 경쟁도(상가) + SNS 여론(블로그) + 검색어 트렌드(네이버) 통합 분석

### 📊 상권 분석 공식 (Scoring Engine)
최종 점수는 다음 4가지 데이터를 각각 25%의 비중으로 합산하여 산출합니다.
> **최종 점수 = (인구 점수 * 0.25) + (수요 점수 * 0.25) + (트렌드 점수 * 0.25) + (경쟁 점수 * 0.25)**

- **인구 점수**: 행안부 데이터를 기반으로 해당 지역의 MZ세대 인구수 점수화
- **수요 점수**: 네이버 검색어 트렌드 API를 통한 키워드 검색량 분석
- **트렌드 점수**: 블로그 크롤링을 통한 감성(긍정/부정) 및 언급량 점수화
- **경쟁 점수**: 소상공인 상가 정보를 통한 업종 밀집도(경쟁 역수) 분석

## 🛠️ 2. 구현된 기술 스택 및 모듈 현황

### 📂 디렉토리 구조
- `core/collectors/`: 4대 데이터 수집기 (Pop, Store, Blog, Naver)
- `core/engine/`: 상권 분석 엔진(`scoring_engine.py`) 및 키워드 생성기(`keyword_gen.py`)
- `data/`: SQLite DB 관리 모듈 (`db_manager.py`, `schema.py`)
- `ui/`: PySide6 기반 프리미엄 다크 모드 UI
- `main.py`: 애플리케이션 통합 실행 파일

### ⚙️ 주요 기능 완료 사항
- [x] SQLite DB 스키마 설계 및 외래 키(FK) 무결성 확보
- [x] 6개 타켓 상권(한남, 이태원, 성수, 연남, 망원, 가로수길) 정의
- [x] 720개 조합의 키워드 자동 생성 로직 구현
- [x] Safe-Mode 일괄 수집기(`main_collector.py`) 구현 (셔플 및 랜덤 지연 적용)
- [x] PySide6 기반 입력/결과 화면 UI 레이아웃 완성
