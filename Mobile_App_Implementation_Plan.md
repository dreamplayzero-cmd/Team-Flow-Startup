# 📱 [공식] Youth Startup Flow 모바일 앱(Flutter) 전환 계획서

본 문서는 기존의 프리미엄 웹 리포팅 툴을 모바일 환경으로 이식하기 위한 최종 구현 계획서입니다. 사용자님의 요청에 따라 **총 6개의 핵심 화면**으로 구성되며, 모바일 환경에서만 가능한 **3D GIS 모델링**을 핵심 가치로 삼습니다.

---

## 1. 프로젝트 개요
*   **앱 이름**: `Youth Startup Flow (flow_startup_app)`
*   **디자인 컨셉**: 럭셔리 다크 모드, 마블 에스테틱, 글래스모피즘
*   **핵심 기술**: Flutter (Google), Mapbox (3D GIS), Provider (상태 관리)

---

## 2. 앱 페이지 구성 (총 6페이지)

웹의 방대한 정보를 모바일의 직관적인 흐름에 맞춰 6개의 독립적인 화면으로 재설계합니다.

| 순서 | 페이지 명칭 | 주요 기능 및 디자인 UX |
| :--- | :--- | :--- |
| **1** | **Login Page** | 첫 진입 화면. 고급 대리석 배경 위로 부드럽게 떠오르는 로그인 폼. |
| **2** | **Main Dashboard** | 개인화된 홈 화면. 최근 분석 리포트 요약과 AI 스코어를 한눈에 확인. |
| **3** | **Persona & Plan Step** | 데이터 입력 화면. 텍스트 입력 대신 단계별(Stepper) 버튼과 슬라이더로 사용자 편의성 극대화. |
| **4** | **3D GIS Scouting Map** | **본 앱의 핵심.** Mapbox를 활용하여 상권 건물을 3D로 시각화하고, 클러스터 밀도를 입체적으로 표현. |
| **5** | **District Leaderboard** | 지역별 유망 상권 순위 리스트. 3D 맵과 연동되어 랭킹별 상세 정보 제공. |
| **6** | **Premium Analysis Report** | 대미를 장식하는 결과 리포트. 3/5/10년 후 시뮬레이션 결과와 AI 전략 보고서를 화려한 애니메이션으로 제공. |

---

## 3. 왜 모바일 3D가 웹보다 우수한가?
1. **네이티브 가속**: 모션이 훨씬 부드럽습니다. 사용자가 맵을 돌릴 때 끊김 없는 60fps 경험을 제공합니다.
2. **햅틱 피드백**: 특정 상권을 터치하거나 3D 건물을 선택할 때 미세한 진동을 주어 "실감 나는" 분석 경험을 제공합니다.
3. **직관적 제어**: 마우스보다 두 손가락을 이용한 3D 카메라 틸트(기울이기) 조작이 훨씬 자연스럽습니다.

---

## 4. 향후 개발 마일스톤
1.  **Phase 1. 기초 환경 설정**: (완료) `flutter create`를 통한 프로젝트 폴더 구축.
2.  **Phase 2. 화면 이동(Routing) 구축**: 6개 페이지를 연결하는 내비게이션 바 제작.
3.  **Phase 3. 3D GIS 엔진 이식**: Mapbox를 이용한 3D 맵 화면 구현.
4.  **Phase 4. 최종 리포트 애니메이션 및 백엔드 연동**.
